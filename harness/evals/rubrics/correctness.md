# Correctness rubric

Score 0-3:

- 0: task is not implemented or violates the requested boundary.
- 1: partial implementation, missing the main acceptance behavior or proof.
- 2: behavior is correct with a small omission in edge coverage or workflow.
- 3: requested behavior, edge cases, and project verification are all proven.

Record the evidence files, tests, and final gate result in the experiment row.
