# Minimality rubric

Score 0-3:

- 0: broad unrelated changes, new architecture, or dependency churn.
- 1: correct direction but substantial unrelated scope or duplication.
- 2: narrow diff with minor cleanup beyond the task.
- 3: smallest coherent diff, reusing existing patterns and no speculative work.

Count unrelated changed files mechanically where possible and explain any
intentional exceptions.
